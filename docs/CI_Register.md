## CI Naming Convention
CI-Type-Name-v<major>.<minor>

Examples:
- DOC-SCMP-v1.0
- SRC-Frontend-v0.1
- SRC-Backend-v0.1

---

## CI Status Definitions
- **Baselined** – Approved and frozen under BL1
- **Under Development** – Can change only via Change Request
- **Released** – Included in a formal release

---

## CI Control Rules
- Any modification to a baselined CI requires an approved Change Request (CR).
- CI versions are updated after approved changes.
